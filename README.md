## ğŸ“¦ Installation

### Prerequisites:
- Node.js installed on your system
- Git installed

### Steps to setup locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/ManojkumarBalini/airport-info-api.git
````

2. Navigate to the project folder:

   ```bash
   cd airport-info-api
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the server:

   ```bash
   npm start
   ```

5. Access the API at:

   ```
   http://localhost:3000
   ```
